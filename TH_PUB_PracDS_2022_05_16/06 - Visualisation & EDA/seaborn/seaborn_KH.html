01. seaborn Package¶

  • a Python data visualization library based on matplotlib

  • provides a high-level interface for drawing attractive and informative statistical graphics

ref: https://seaborn.pydata.org/

02. B. Violin plots¶

  • combination of boxplot and estimate of the underlying distribution
    • width of a violin plot is the frequency (ie., how common) a value is within the data

Compare custom classifications¶

In [30]:
def age_class(age):
    if age < 10:
        return 'under 10'
    elif age < 16:
        return 'teens under 16'
    elif age < 18:
        return 'teens over 16'
    else:
        return 'adult'
In [31]:
sns.violinplot(data=df, y='Height', x=df['Age'].map(age_class));
